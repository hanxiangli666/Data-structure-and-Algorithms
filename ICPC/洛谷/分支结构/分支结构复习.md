# 分支结构题目复习

分支结构核心：**判断条件 → 选择执行路径**。难点在于边界处理、条件覆盖完整性、特殊情况的提前排除。

---

## 一、简单条件判断

### 1. apple（单复数）
**题意：** x 个苹果，0或1个输出 `apple.`，否则输出 `apples.`

```cpp
if (x == 0 || x == 1) cout << x << " apple.";
else                   cout << x << " apples.";
```
**要点：** 注意 `0` 也是单数形式，不要漏掉。

---

### 2. 闰年
**题意：** 判断是否为闰年，输出 `1` 或 `0`。

```cpp
bool leap = (n % 4 == 0 && n % 100 != 0) || (n % 400 == 0);
cout << leap;
```
**闰年规则：** 能被4整除且不能被100整除，**或**能被400整除。

---

### 3. 数的性质
**题意：** 给定整数 x，判断 p1（偶数）和 p2（4 < x ≤ 12）的四种逻辑组合。

```cpp
bool p1 = (x % 2 == 0);
bool p2 = (x > 4) && (x <= 12);
cout << (p1 && p2) << " " << (p1 || p2) << " " << (p1 != p2) << " " << (!p1 && !p2);
```
**要点：** `p1 != p2` 等价于异或（恰好一个为真）；`!p1 && !p2` 等价于两者均为假。

---

### 4. 系统
**题意：** 比较本地系统 `5n` 和洛谷系统 `3n+11` 的大小，输出较小的一方。

```cpp
if (5*n < 3*n+11) cout << "Local";
else              cout << "Luogu";
```
**要点：** 化简得 `2n < 11`，即 `n ≤ 5` 时本地更小，否则洛谷更小。直接写条件即可。

## 二、分段/区间判断

### 5. 电费（分段计费）
**题意：** 用电量 n 度，按三段阶梯价格计算费用（保留1位小数）。

| 区间 | 单价 |
|------|------|
| ≤ 150 度 | 0.4463 |
| 151–400 度 | 0.4663 |
| > 400 度 | 0.5663 |

```cpp
if (n <= 150)
    cost = n * 0.4463;
else if (n <= 400)
    cost = 150*0.4463 + (n-150)*0.4663;
else
    cost = 150*0.4463 + 250*0.4663 + (n-400)*0.5663;
printf("%.1f\n", cost);
```
**要点：** 每段只计算**超出部分**，前段费用固定叠加；用 `printf("%.1f")` 输出浮点数。

### 6. 肥胖问题（BMI）
**题意：** 输入体重 m（kg）和身高 h（m），计算 BMI = m/h²，分三档输出。

```cpp
double bmi = m / (h * h);
if      (bmi < 18.5)  cout << "Underweight";
else if (bmi < 24)    cout << "Normal";
else {
    cout << bmi << "\n" << "Overweight";  // 超重时先输出bmi值
}
```
**要点：** Overweight 时需额外输出 bmi 值，且顺序是先 bmi 后 "Overweight"。

## 三、排序 + 映射

### 7. 三位数排序
**题意：** 输入三个数，从小到大输出。

```cpp
int arr[3] = {a, b, c};
sort(arr, arr+3);
cout << arr[0] << " " << arr[1] << " " << arr[2];
```

### 8. ABC
**题意：** 输入三个数，排序后令最小=A、次小=B、最大=C，按给定顺序字符串输出对应值。

```cpp
sort(arr, arr+3);  // arr[0]=A, arr[1]=B, arr[2]=C
for (int i = 0; i < 3; i++)
    cout << arr[order[i] - 'A'];  // 字符转下标：'A'→0, 'B'→1, 'C'→2
```
**要点：** `order[i] - 'A'` 将字符映射为数组下标，是字符索引的常用技巧。

## 四、三角形判断

### 9. 三角形
**题意：** 判断三边能否构成三角形，若能判断是直角/锐角/钝角，以及是否等腰/等边。

```cpp
// 第一步：能否构成三角形（严格大于）
if (a+b <= c || b+c <= a || a+c <= b) { cout << "Not triangle"; return 0; }

// 第二步：角的类型（用平方和比较，避免浮点）
int maxSide = max({a,b,c});
int sumSq = a*a + b*b + c*c - maxSide*maxSide;
if      (sumSq == maxSide*maxSide) cout << "Right triangle";
else if (sumSq >  maxSide*maxSide) cout << "Acute triangle";
else                               cout << "Obtuse triangle";

// 第三、四步：等腰/等边（独立 if，可能同时输出）
if (a==b || b==c || a==c) cout << "Isosceles triangle";
if (a==b && b==c)         cout << "Equilateral triangle";
```
**关键技巧：**
- 用整数平方比较，避免 `sqrt` 浮点误差
- 等腰和等边是**独立 if**，因为等边三角形同时也是等腰三角形，两行都要输出

---

### 10. 三角函数
**题意：** 给定直角三角形三边，求最短直角边与斜边之比（化最简分数）。

```cpp
long long hyp = max({a, b, c});  // 斜边（最大边）
long long leg = min({a, b, c});  // 最短直角边
long long g = __gcd(leg, hyp);
cout << leg/g << "/" << hyp/g;
```
**要点：** 用 `__gcd` 求最大公约数约分；题目保证是直角三角形，取最大和最小即可。

## 五、数学建模

### 11. 苹果和虫子
**题意：** m 个苹果，虫子每 t 小时吃一个，已经吃了 s 小时，问剩几个。

```cpp
if (t == 0) { cout << 0; }  // 特殊情况：t=0 时整除会崩溃
else {
    int eaten = s / t + (s % t != 0 ? 1 : 0);  // 向上取整
    cout << max(0, m - eaten);  // 不能为负数
}
```
**要点：**
- 先排除 `t == 0` 的特殊情况（防止除零）
- 吃掉的数量**向上取整**（吃了一半也算一个）
- 结果不能小于 0，用 `max(0, ...)`

---

### 12. 买铅笔
**题意：** 需要 n 支铅笔，三种包装（数量、单价），求最少花费。

```cpp
int ans = INT_MAX;
for (int i = 0; i < 3; i++) {
    int cnt, price;
    cin >> cnt >> price;
    int packs = (n + cnt - 1) / cnt;  // 向上取整（必须买整包）
    ans = min(ans, price * packs);
}
```
**向上取整公式：** `(n + cnt - 1) / cnt` ← 整数向上取整的标准写法

## 六、字符串处理

### 14. ISBN
**题意：** 验证 ISBN 校验码，格式 `X-XXX-XXXXX-X`，校验位 = (∑i×dᵢ) % 11，10→`X`。

```cpp
int sum = 0, weight = 1;
for (int i = 0; i < s.size(); i++) {
    if (s[i] == '-') continue;     // 跳过连字符
    if (weight > 9) break;          // 只取前9位参与计算
    sum += (s[i] - '0') * weight++;
}
int check = sum % 11;
char correct = (check == 10) ? 'X' : ('0' + check);

if (s.back() == correct) cout << "Right";
else { /* 输出前面部分 + correct */ }
```
**要点：**
- 跳过 `-` 用 `continue`
- `weight > 9` 时停止（第10位是校验位，不参与计算）
- `s.back()` 取最后一个字符

## 七、最值查找

### 15. 不开心
**题意：** 7天，每天读 a 小时书 + b 小时玩，超出8小时的部分为不开心值，找最大不开心那天。

```cpp
int maxUnhappy = 0, ans = 0;
for (int i = 1; i <= 7; i++) {
    int unhappy = max(0, (a + b) - 8);
    if (unhappy > maxUnhappy) {  // 严格大于，平局取前面那天
        maxUnhappy = unhappy;
        ans = i;
    }
}
```
**要点：** 用**严格大于**而非 `>=`，平局保留较早的天（题目要求）。

## 八、循环 + 分支

### 16. 小鱼的航程
**题意：** 从星期 x 出发，连续游 n 天，工作日（周一到周五）每天 250km，求总里程。

```cpp
int day = x;
for (int i = 0; i < n; i++) {
    if (day < 6) total += 250;    // 1~5 = 周一到周五
    day = day % 7 + 1;            // 循环到下一天（7→1）
}
```
**要点：** `day % 7 + 1` 是"星期循环"的标准写法（7%7=0，0+1=1，即周日→周一）。

## 分支结构通用思路

```
读入数据
  ↓
有无需要提前排除的特殊情况？（如除零、无效输入）
  ↓ 是→先处理特殊情况（return / 输出后退出）
主体分支判断
  ├─ if-else if-else：互斥情况（三角形类型、BMI区间、电费阶梯）
  └─ 独立 if：可叠加情况（三角形同时是等腰又是直角）
  ↓
输出结果
```

## 常用技巧速查

| 场景 | 写法 |
|------|------|
| 向上取整（整数）| `(n + k - 1) / k` |
| 星期循环 | `day = day % 7 + 1` |
| 字符映射下标 | `c - 'A'`（A→0, B→1, C→2）|
| 避免浮点误差 | 改用整数平方比较（三角形类型）|
| 最大公约数 | `__gcd(a, b)`（需 `<algorithm>`）|
| 防止结果为负 | `max(0, expr)` |
| 特殊情况前置 | 先 `if (特殊) { return; }` 再写正常逻辑 |

## 易错点

1. **边界条件**：`<=` vs `<`，区间端点归属哪段（如电费的 150、400）
2. **独立 if vs if-else**：多个标签可同时成立时必须用独立 if
3. **特殊情况遗漏**：苹果虫子的 `t==0`、三角形不合法提前退出
4. **整数除法**：需要浮点结果时记得加 `.0`（如 `500.0/3`）
5. **平局处理**：最值更新用严格 `>` 还是 `>=` 取决于题意
