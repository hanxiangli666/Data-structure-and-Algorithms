# 函数题目复习

函数核心：**封装重复逻辑、递归分解问题、结构体组织数据**。难点在于递归终止条件和结构体成员函数的设计。

## 一、工具函数封装

### 1. 质数判断（isPrime）
**题意：** 从 n 个数中筛出所有质数输出。

```cpp
bool isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i*i <= n; i++)  // 只枚举到 √n
        if (n % i == 0) return false;
    return true;
}
```
**输出格式控制（数字间空格，末尾无空格）：**
```cpp
bool first = true;
for (int i = 0; i < n; i++)
    if (isPrime(a[i])) { if (!first) cout<<" "; cout<<a[i]; first=false; }
```

### 2. 线性筛质数（埃氏筛）
**题意：** 筛出 [2, MAXN) 内所有质数，用于哥德巴赫猜想。

```cpp
bool isPrime[MAXN];
void sieve() {
    fill(isPrime, isPrime+MAXN, true);
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i < MAXN; i++)
        if (isPrime[i])
            for (long long j = (long long)i*i; j < MAXN; j += i)
                isPrime[j] = false;
}
```
**对比：**

| 方式 | 时间复杂度 | 适用场景 |
|------|-----------|---------|
| 逐个判断 `isPrime(n)` | O(√n) 每次 | 单次或少量判断 |
| 埃氏筛 `sieve()` | O(n log log n) 预处理 | 大量查询质数 |

**题意：** 给定三角形三顶点，求周长（保留2位小数）。

double dist(double x1, double y1, double x2, double y2) {

### 5. 歌唱比赛（去极值求平均）
**题意：** n 名选手各有 m 个评分，去最高最低后求平均，输出最高分（保留2位小数）。

```cpp
double calcScore(vector<int>& scores) {
    int total=0, maxVal=scores[0], minVal=scores[0];
    for (int s : scores) { total+=s; maxVal=max(maxVal,s); minVal=min(minVal,s); }
    return (double)(total - maxVal - minVal) / (scores.size() - 2);
}
// 输出
cout << fixed << setprecision(2) << best;
```
**要点：** 函数参数用 `vector<int>&` 引用传递，避免拷贝；`scores.size()-2` 注意 `size()` 返回无符号数，减法可能溢出，建议强转 `(int)scores.size()`。

## 二、递归

### 递归三要素

```
1. 终止条件：什么情况下直接返回（不再递归）
2. 递推关系：当前问题如何依赖子问题
3. 返回值：子问题的结果如何组合成当前问题的结果
```

---

### 6. 计算阶乘
**题意：** 计算 n!

```cpp
int factorial(int n) {
    if (n == 1) return 1;           // 终止条件
    return n * factorial(n - 1);   // 递推关系
}
```

### 7. 猴子吃桃
**题意：** 猴子每天吃掉当天桃子的一半再多一个，第 n 天剩1个，求第1天有多少个。

```cpp
int peach(int day, int n) {
    if (day == n) return 1;                  // 终止条件：第n天剩1个
    return (peach(day+1, n) + 1) * 2;       // 逆推：今天 = (明天+1) × 2
}
// 调用
cout << peach(1, n);
```
**逆推思路：** 设明天有 x 个，则今天有 x 个时吃掉 `today/2 + 1 = x`，即 `today = (x+1)*2`。

### 8. 赦免战俘（分治递归）
**题意：** 2ⁿ × 2ⁿ 的方阵初始全为1，递归地将左上角四分之一全置0，对其余三块继续递归。

```cpp
void solve(int r, int c, int size) {
    if (size == 1) return;  // 单格，无法再分
    int half = size / 2;
    // 左上角全置0（赦免）
    for (int i=r; i<r+half; i++)
        for (int j=c; j<c+half; j++)
            grid[i][j] = 0;
    // 递归处理其余三个子矩阵
    solve(r,      c+half, half);  // 右上
    solve(r+half, c,      half);  // 左下
    solve(r+half, c+half, half);  // 右下
}
// 2^n 用位运算
int size = 1 << n;  // 等价于 pow(2,n)，但更快
```
**要点：** 分治递归的关键是确定每次子问题的**起点坐标**和**边长**；`1 << n` 是 2ⁿ 的位运算写法。

### 9. 最厉害的学生
**题意：** n 个学生，找总分最高的（相同总分保留先读到的）。

```cpp
struct Student {
    string name; int chinese, math, english;
    int total() const { return chinese + math + english; }
};

Student best; int bestTotal = -1;
for (int i=0; i<n; i++) {
    Student s; cin >> s.name >> s.chinese >> s.math >> s.english;
    if (s.total() > bestTotal) { bestTotal=s.total(); best=s; }  // 严格>，相同保留前者
}
```

### 10. 培训
**题意：** 每个学生经过培训后年龄+1，成绩×1.2（不超过600），用函数封装。

```cpp
Student train(Student s) {  // 值传递，不修改原对象
    s.age += 1;
    s.score = min(s.score * 6 / 5, 600);  // 整数运算：×6/5 = ×1.2（避免浮点）
    return s;
}
```
**要点：** `score * 6 / 5` 用整数运算实现 ×1.2，避免浮点误差；值传递函数不影响原始数据。

### 11. 评等级
**题意：** 综合分 = 学业×0.7 + 素质×0.3，判断是否优秀（总分>140 且 综合分≥80）。

```cpp
struct Student {
    int id, academic, quality;
    int total() const { return academic + quality; }
};

bool isExcellent(const Student& s) {
    bool totalOk = s.total() > 140;
    // 避免浮点：0.7×academic + 0.3×quality >= 80  两边×10
    bool scoreOk = s.academic*7 + s.quality*3 >= 800;
    return totalOk && scoreOk;
}
```
**要点：** 浮点比较 `>= 80.0` 改为整数比较 `×10 >= 800`，消除精度误差；`const Student&` 引用传参效率高且不可修改。

## 四、数学推导

### 13. 集合求和
**题意：** n 个数，求其所有非空子集的元素和之总和。

**数学结论：** 每个元素在 2ⁿ 个子集中各出现 2ⁿ⁻¹ 次（含它/不含它各一半）。

```cpp
while (cin >> x) { sum += x; n++; }
cout << sum * (1LL << (n-1));  // sum × 2^(n-1)
```
**要点：** `1LL << (n-1)` 是 long long 类型的 2ⁿ⁻¹，防止 n 较大时溢出 int。

## 函数通用模板速查

### 函数参数传递方式

| 方式 | 写法 | 特点 |
|------|------|------|
| 值传递 | `void f(int x)` | 函数内修改不影响外部 |
| 引用传递 | `void f(int& x)` | 可修改外部变量 |
| 常量引用 | `void f(const Student& s)` | 不可修改，避免拷贝（大对象用）|

### 输出格式

| 需求 | 写法 |
|------|------|
| 保留n位小数 | `cout << fixed << setprecision(n) << x`（需 `<iomanip>`）|
| printf 保留n位 | `printf("%.nf", x)` |
| 数字间空格无尾空格 | `bool first=true; if(!first) cout<<" "; first=false;` |

### 常用位运算

| 含义 | 写法 |
|------|------|
| 2ⁿ | `1 << n` 或 `1LL << n`（大数用 long long）|
| 判断第k位是否为1 | `(x >> k) & 1` |

## 易错点

1. **递归终止条件**：忘写或写错导致无限递归，务必先想清楚"什么时候不再递归"
2. **值传递 vs 引用传递**：结构体作参数时，若不需要修改用 `const&`，需要修改用 `&`，只读小对象可值传递
3. **整数溢出**：`i*i` 或 `1<<n` 在 n 较大时超出 int，改用 `(long long)i*i` 或 `1LL<<n`
4. **浮点比较**：`>= 80.0` 可能因精度问题出错，乘以整数倍改为整数比较
5. **`size()` 减法**：`vector.size()-2` 当 size<2 时会下溢（无符号数），需强转为 `(int)`
6. **成员函数加 `const`**：不修改成员变量的函数应加 `const`，否则无法用于 `const` 对象
7. **`fixed` 和 `setprecision` 要同时用**：单独用 `setprecision(2)` 是总有效数字位数，不是小数位数
