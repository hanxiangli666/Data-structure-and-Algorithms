# 顺序结构题目复习

顺序结构：程序从上到下逐行执行，无分支、无循环，核心在于**理清计算步骤和单位换算**。

---

## 题目汇总

### 1. 成绩（加权平均）

**题意：** 给定平时分 a、期中分 b、期末分 c，按 20%、30%、50% 的权重求总评成绩（取整）。

**关键点：**

- 整数运算会截断小数，`a * 0.2` 会隐式转换为 `double`，最终赋值给 `int` 时截断
- 题目要求输出整数，直接用 `int` 接收即可（截断 = 向下取整）

```cpp
int total = a * 0.2 + b * 0.3 + c * 0.5;
```

**模板：** 加权求和 → 直接输出

---

### 2. 大象喝水（圆柱体积 + 向上取整）

**题意：** 圆柱形水桶，高 h、半径 r（单位：分米），求需要多少桶水才能装满 20000 升。

**关键点：**

- 体积公式：`V = π × r² × h`，π 取 3.14
- 单位换算：1 分米³ = 1 升，无需换算
- **向上取整**：用 `ceil()`，需包含 `<cmath>`，且参数要用浮点数 `20000.0 / v`

```cpp
double v = 3.14 * r * r * h;
int bottle = ceil(20000.0 / v);
```

**模板：** 几何公式 → 向上取整（`ceil`）

---

### 3. 小鱼的游泳时间（时间差计算）

**题意：** 给定开始时刻（a 时 b 分）和结束时刻（c 时 d 分），求游泳时长（小时 + 分钟）。

**关键点：**

- 统一换算成分钟再做差，避免借位问题
- 再拆分回小时和分钟

```cpp
int total_min = (c * 60 + d) - (a * 60 + b);
int hour = total_min / 60;
int min  = total_min % 60;
```

**模板：** 时间/单位 → 统一换算 → 计算 → 拆回原单位

---

### 4. 小玉买文具（进制换算 + 整除）

**题意：** 给定 a 元 b 角，铅笔每支 1 角 9 分（即 1.9 角），求最多能买几支。

**关键点：**

- 统一换算成"角"：`total = a * 10 + b`
- 每支价格 1.9 角 → 换算成"分"避免浮点：1 角 = 10 分，每支 19 分，总钱 `(a*10+b)*10` 分
  - **本题用角直接整除也可**：`(a*10+b) / 19`（1.9 角 × 10 = 19 角分混合？）
  - 实际上代码用 `total_mon / 19`，其中 `total_mon = a*10 + b`（单位：角），19 对应 1.9 角×10，**相当于都放大了 10 倍**，结果正确

```cpp
int total_mon = a * 10 + b;  // 单位：角
int ans = total_mon / 19;    // 1.9角 × 10 = 19，放大10倍统一计算
```

**模板：** 含小数的金额 → 放大为整数单位 → 整除

## 顺序结构通用思路

```
读入数据
  ↓
单位换算（统一到同一单位）
  ↓
按公式计算
  ↓
处理取整方式（截断 / 向上 ceil / 向下 floor / 四舍五入 round）
  ↓
输出结果
```

## 常用函数速查

| 需求     | 函数           | 头文件      |
| -------- | -------------- | ----------- |
| 向上取整 | `ceil(x)`    | `<cmath>` |
| 向下取整 | `floor(x)`   | `<cmath>` |
| 四舍五入 | `round(x)`   | `<cmath>` |
| 整数截断 | 赋值给 `int` | —          |

## 易错点

1. **整数除法截断**：`7 / 2 = 3`，不是 3.5，需要浮点结果时至少一个操作数用 `double`
2. **`ceil` 参数类型**：`ceil(20000 / v)` 若 `20000` 是 `int` 且 `v` 是 `double`，结果是 `double`，正确；但最好写 `20000.0`
3. **含小数的价格**：优先放大为整数（乘以10/100）再做整除，避免浮点精度误差
