# 字符串题目复习

字符串核心：**字符与数字互转、逐字符遍历、子串查找与操作**。难点在于读取方式的选择和边界处理。

---

## 一、字符串读取

### 读取方式对比

| 场景             | 写法                            | 注意                           |
| ---------------- | ------------------------------- | ------------------------------ |
| 单词（无空格）   | `cin >> s`                    | 遇空格停止                     |
| 整行（含空格）   | `getline(cin, s)`             | 读到换行符为止（换行符被丢弃） |
| cin 后接 getline | `cin.ignore()` 跳过残留换行符 | 否则 getline 读到空行          |
| 逐行读多行       | `while(getline(cin, line))`   | EOF 时退出                     |

### 1. 标题统计

**题意：** 统计标题中非空格字符数（含汉字，需读整行）。

```cpp
getline(cin, s);
for (char c : s) if (c != ' ') cnt++;
```

### 2. 口算练习题（stringstream 解析）

**题意：** n 道口算题，每行可能以运算类型字母开头，解析出操作数和运算符，输出算式及其长度。

```cpp
cin >> n;
cin.ignore();  // 跳过换行符，否则第一次 getline 读到空行
char op = 'a';
for (int i = 0; i < n; i++) {
    string line; getline(cin, line);
    stringstream ss(line);
    string first; ss >> first;
    int a, b;
    if (first == "a" || first == "b" || first == "c") {
        op = first[0]; ss >> a >> b;  // 本行指定了运算类型
    } else {
        a = stoi(first); ss >> b;     // 沿用上次的运算类型
    }
    // 输出算式和长度
    string expr = to_string(a) + sign + to_string(b) + "=" + to_string(result);
    cout << expr << "\n" << expr.size() << "\n";
}
```

**要点：**

- `cin.ignore()` 消耗 `cin>>n` 留下的换行符
- `stringstream ss(line)` 将一行字符串包装成流，方便用 `>>` 逐词解析
- `stoi(s)` 将字符串转为整数

for (char& c : s) c = toupper(c);

### 4. 凯撒密码

## 三、字母频率统计

### 5. 笨小猴

**题意：** 统计单词中每个字母出现次数，判断（最大频率 - 最小频率）是否为质数。

```cpp
int cnt[26] = {};
for (char c : s) cnt[c - 'a']++;

int maxn = 0, minn = INT_MAX;
for (int i = 0; i < 26; i++)
    if (cnt[i] > 0) { maxn = max(maxn, cnt[i]); minn = min(minn, cnt[i]); }

int diff = maxn - minn;
cout << (isPrime(diff) ? "Lucky Word\n" : "No Answer\n") << (isPrime(diff) ? diff : 0);
```

**要点：** `cnt[i] > 0` 只统计出现过的字母；minn 初始化为 `INT_MAX`（而非0）。

### 6. 垂直柱状图

**题意：** 读4行文本，统计大写字母频率，绘制柱状图（按行从高到低输出，末尾到最后一个有值的列）。

```cpp
// 统计
for (char c : line) if (c>='A' && c<='Z') cnt[c-'A']++;
// 按行输出：每行从高到低
for (int row = maxCnt; row >= 1; row--) {
    // 找本行最后一个 >= row 的字母列，截断尾部空白
    int last = -1;
    for (int i = 25; i >= 0; i--) if (cnt[i] >= row) { last=i; break; }
    for (int i = 0; i <= last; i++) cout << (i>0?" ":"") << (cnt[i]>=row?"*":" ");
    cout << "\n";
}
// 字母行
for (int i=0; i<26; i++) cout << (i>0?" ":"") << (char)('A'+i);
```

**要点：** 每行末尾不能有多余空格，要找到该行最后一个有柱子的列（`last`）才能截断。

## 四、字符串操作 API

### 7. 文字处理软件

**题意：** 对字符串执行4种操作：末尾追加、截取子串、插入子串、查找子串。

```cpp
s += str;                    // 操作1：追加
s = s.substr(a, b);          // 操作2：从第a个字符起取b个（下标0开始）
s.insert(a, str);            // 操作3：在第a个字符前插入
int pos = s.find(str);       // 操作4：查找，返回下标或 string::npos
```

**`string` 常用 API 速查：**

| 函数                            | 说明                                           |
| ------------------------------- | ---------------------------------------------- |
| `s.size()` / `s.length()`   | 字符串长度                                     |
| `s.substr(pos, len)`          | 从 pos 起取 len 个字符                         |
| `s.find(t, start)`            | 从 start 起查找 t，返回位置或 `string::npos` |
| `s.insert(pos, t)`            | 在 pos 前插入字符串 t                          |
| `s += t`                      | 末尾追加                                       |
| `s.back()`                    | 最后一个字符                                   |
| `reverse(s.begin(), s.end())` | 反转（需 `<algorithm>`）                     |

### 8. 统计单词数

**题意：** 在文章中查找某单词（独立单词，前后是空格或边界），统计出现次数和第一次位置。

```cpp
// 全部转小写
int start = 0;
while (true) {
    int pos = article.find(word, start);
    if (pos == string::npos) break;
    bool leftOK  = (pos == 0 || article[pos-1] == ' ');
    bool rightOK = (pos+wLen == aLen || article[pos+wLen] == ' ');
    if (leftOK && rightOK) { cnt++; if (firstPos==-1) firstPos=pos; }
    start = pos + 1;  // 移动到下一个位置继续找
}
```

**要点：** 验证左右边界均为空格或字符串端点，防止匹配到单词中间；`start = pos+1` 而非 `pos+wLen` 防止漏掉相邻匹配。

### 9. 数字反转（升级版）

**题意：** 对整数/小数/分数/百分数分别做反转处理，去除前导零和尾部零。

```cpp
// 类型识别
if (s.back() == '%')          // 百分数
else if (s.find('/') != npos) // 分数
else if (s.find('.') != npos) // 小数
else                          // 整数

// 整数部分反转（去前导零）
reverse(s.begin(), s.end());
while (start < s.size()-1 && s[start]=='0') start++;  // 至少保留1位

// 小数部分反转（去前导零 + 去尾部零）
// 先去尾部零，再去前导零（或反转后去两端）
```

**要点：** 小数部分反转后需去尾部零（原来是前导零）；字符串分割用 `s.find('.')` 找分隔符位置，`s.substr(0, pos)` 和 `s.substr(pos+1)` 分别取两段。

## 五、字母-数字映射

### 10. 手机键盘

**题意：** 九宫格手机，输入句子，计算总按键次数（空格按0键1次，字母按对应键若干次）。

```cpp
string keys[] = {"abc","def","ghi","jkl","mno","pqrs","tuv","wxyz"};
int cost[26] = {};
for (int i=0; i<8; i++)
    for (int j=0; j<keys[i].size(); j++)
        cost[keys[i][j]-'a'] = j+1;  // 该字母在组内是第j+1个，需按j+1次

getline(cin, s);  // 含空格用getline
for (char c : s)
    total += (c==' ') ? 1 : cost[c-'a'];
```

**要点：** 预处理建映射表；空格对应0键按1次。

### 11. 你的飞碟在这儿

**题意：** 两个字符串，每个字母映射为 A→1, B→2, ..., Z→26，求各字母值之积对47取模，相等输出GO否则STAY。

```cpp
long long pa=1, pb=1;
for (char c : a) pa *= (c-'A'+1);
for (char c : b) pb *= (c-'A'+1);
cout << (pa%47 == pb%47 ? "GO" : "STAY");
```

**要点：** 乘积可能很大，用 `long long`；取模后再比较。

### 12. 斯诺登的密码

**题意：** 句子中的数字单词（zero~twenty及部分词）转为"值²%100"的两位字符串，拼接后的最小数字。

```cpp
// 对每个两位字符串，排序规则：a+b < b+a（字典序最小拼接）
sort(nums.begin(), nums.end(), [](const string& a, const string& b) {
    return a+b < b+a;
});
// 拼接后去前导零
```

**要点：** 最小数字拼接问题：比较 a+b 和 b+a 的字典序，不能直接按字符串大小排序。

## 六、模拟 / 特殊规则

### 13. 语句解析

**题意：** 输入多条 "x:=y;" 格式的赋值语句，y 是数字或变量名，模拟执行后输出 a, b, c 的值。

```cpp
for (int i = 0; i < s.size(); i += 5) {  // 每条语句固定5字符
    int left = s[i] - 'a';    // 左边变量下标
    char right = s[i+3];      // 右边值
    if (right >= '0' && right <= '9') val[left] = right - '0';
    else                               val[left] = val[right-'a'];
}
```

**要点：** 每条语句长度固定（`a:=x;` = 5字符），直接用 `i += 5` 步进；判断右边是数字还是变量。

### 14. 小果的键盘

**题意：** 长度为 n 的 V/K 字符串，恰好修改一个字符（或不修改），最大化 "VK" 出现次数。

```cpp
int countVK(string s) { /* 统计相邻VK对数 */ }

int ans = countVK(s);  // 不修改的情况
for (int i=0; i<n; i++) {
    string t = s;
    t[i] = (s[i]=='V') ? 'K' : 'V';  // 翻转第i个字符
    ans = max(ans, countVK(t));
}
```

**要点：** 枚举每个位置，翻转后重新统计；时间复杂度 O(n²)，n≤100 可行。

### 15. 单词覆盖还原

**题意：** 字符串被部分覆盖，统计可能含 "boy" 的位置数和可能含 "girl" 的位置数。

```cpp
// 每个位置i，检查是否能形成boy/girl
// boy: s[i]=='b' || s[i+1]=='o' || s[i+2]=='y'
// girl: s[i]=='g' || s[i+1]=='i' || s[i+2]=='r' || s[i+3]=='l'
```

**要点：** 条件是 `||`（任意一位匹配即可能），不是 `&&`（因为其他位可能被覆盖）；注意循环边界不越界。

## 字符串通用技巧速查

| 场景             | 写法                                                          |
| ---------------- | ------------------------------------------------------------- |
| 字符转下标       | `c - 'a'`（0~25），`c - 'A'`（0~25），`c - '0'`（0~9） |
| 下标转字符       | `(char)('a' + i)`，`(char)('0' + d)`                      |
| 转大写/小写      | `toupper(c)` / `tolower(c)`                               |
| 字母循环偏移     | `'a' + (c - 'a' + n) % 26`                                  |
| 字符串转整数     | `stoi(s)`（自动去前导零）                                   |
| 整数转字符串     | `to_string(x)`                                              |
| 查找子串         | `s.find(t)` 返回下标或 `string::npos`                     |
| 截取子串         | `s.substr(pos, len)`                                        |
| 反转字符串       | `reverse(s.begin(), s.end())`                               |
| 读整行（含空格） | `getline(cin, s)`                                           |
| cin后用getline   | 先调用 `cin.ignore()`                                       |
| 字符串流解析     | `stringstream ss(line); ss >> word;`                        |

## 易错点

1. **cin 和 getline 混用**：`cin >> n` 后紧跟 `getline` 会读到空行，必须先 `cin.ignore()`
2. **string::npos**：`find` 失败返回 `string::npos`（值为 `-1` 的无符号数），判断时用 `== string::npos`，不要和 `-1` 比较
3. **substr 参数**：`substr(pos, len)` 第二个参数是**长度**，不是结束位置
4. **字母频率初始化**：`minn` 初始化为 `INT_MAX`，不能初始化为 0（否则没出现过的字母会把 minn 置为0）
5. **柱状图尾部空格**：每行末尾不能多输出空格，需要找最后一个有值的列
6. **最小数字拼接排序**：不能直接字典序排序字符串，要比较 `a+b` 和 `b+a`
7. **单词边界验证**：用 `find` 查找单词时，需验证左右是空格或字符串端点，否则会匹配到单词内部
