# 循环结构题目复习

循环结构核心：**确定循环变量、终止条件、每轮做什么**。难点在于何时用 `for` 还是 `while`，以及边界和循环后收尾。

---

## 一、简单累加 / 求最值

### 1. 求和

**题意：** 求 1 到 n 的和。

```cpp
for (int i = 1; i <= n; i++) sum += i;
// 或公式：sum = n*(n+1)/2
```

---

### 2. 找最小值

**题意：** n 个数，找最小值。

```cpp
int min_val = 1e9;  // 初始设极大值
for (int i = 0; i < n; i++) { int x; cin>>x; if (x < min_val) min_val = x; }
```

**模板：** 初始化为极大/极小值 → 逐个比较更新。

---

### 3. 极差

**题意：** n 个数，输出最大值与最小值之差。

```cpp
sort(arr, arr+n);
cout << arr[n-1] - arr[0];
```

**要点：** 排序后直接取首尾之差；也可单次循环同时维护 max/min。

---

### 4. 打分

**题意：** n 个评分，去掉最高分和最低分，求剩余平均（保留2位小数）。

```cpp
int sum=0, maxVal=0, minVal=10;
for (int i=0; i<n; i++) { cin>>x; sum+=x; maxVal=max(maxVal,x); minVal=min(minVal,x); }
printf("%.2f\n", (double)(sum - maxVal - minVal) / (n - 2));
```

**要点：** 一次循环同时维护 sum、max、min，最后统一计算。

---

### 5. 分类平均

**题意：** 1 到 n 中，能被 k 整除的为 A 组，其余为 B 组，分别求平均（保留1位小数）。

```cpp
for (int i=1; i<=n; i++) {
    if (i % k == 0) { sumA += i; cntA++; }
    else            { sumB += i; cntB++; }
}
printf("%.1f %.1f\n", (double)sumA/cntA, (double)sumB/cntB);
```

## 二、条件循环（while）

### 6. 级数求和

**题意：** 求最小的 n，使 1 + 1/2 + 1/3 + ... + 1/n > k。

```cpp
double sum = 0; int n = 0;
while (sum <= k) { n++; sum += 1.0/n; }
cout << n;
```

**要点：** 先加后判还是先判后加要看清题意；`1.0/n` 避免整数除法。

---

### 7. 一尺之棰

**题意：** 每天减半，问几天后长度 ≤ 1（初始为 a，用整数除法）。

```cpp
int day = 1;
while (a > 1) { day++; a /= 2; }  // 整数除法自动向下取整
cout << day;
```

**要点：** 第1天已存在，所以 `day` 从 1 开始。

---

### 8. 游泳

**题意：** 每次游泳距离 = 上次 × 0.98，第一次 2 米，累计直到超过 s 米，问需几次。

```cpp
double sum=0, dist=2.0; int steps=0;
while (sum < s) { sum += dist; steps++; dist *= 0.98; }
cout << steps;
```

---

### 9. 金币

**题意：** 第1天1枚，连续1天；第2天2枚，连续2天；第i天i枚，连续i天...共 k 天，求总金币。

```cpp
int stage=1;
while (k > 0) {
    if (k >= stage) { sum += (long long)stage*stage; k -= stage; }
    else            { sum += (long long)stage*k; k=0; }
    stage++;
}
```

**要点：** 每阶段贡献 `stage × stage` 枚（连续 stage 天每天 stage 枚）；最后一段可能不满，单独处理。

---

### 10. 津津的存储计划

**题意：** 每月领 300 元，给妈妈整百元（手头剩的整百数），自留零头，年末妈妈按1.2倍返还，统计年末总钱数。若某月不够花则输出负数月份。

```cpp
for (int i=0; i<12; i++) {
    hand += 300;
    if (hand < budget[i]) { cout << -(i+1); return 0; }
    int remain = hand - budget[i];
    int give = (remain/100)*100;  // 整百部分给妈妈
    hand = remain % 100;          // 零头自留
    save += give;
}
double total = save * 1.2 + hand;
cout << total;
```

**要点：** `(remain/100)*100` 取整百；最终妈妈的钱 × 1.2 后加上自己手头的零头。

## 三、质数相关

### 11. 质数判断（模板）

```cpp
bool isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i*i <= n; i++)  // 只需枚举到√n
        if (n % i == 0) return false;
    return true;
}
```

**要点：** `i*i <= n` 等价于 `i <= sqrt(n)`，避免浮点，效率 O(√n)。

---

### 12. 质数口袋

**题意：** 从 2 开始依次放入质数，直到再放下一个质数就会超过 L，输出所有放入的质数和个数。

```cpp
for (int i=2; sum+i<=L; i++)  // 关键：循环条件是"放入后不超过L"
    if (isPrime(i)) { cout<<i; sum+=i; cnt++; }
cout << cnt;
```

**要点：** 循环条件写 `sum + i <= L`（放入当前质数后不超限），而非先放再判。

---

### 13. 较大质数

**题意：** 给定合数 n，找它的较大质因子（n = 较小质因子 × 较大质因子）。

```cpp
for (long long i=2; i*i<=n; i++)
    if (n % i == 0) { cout << n/i; break; }  // i是最小因子，n/i是较大的
```

**要点：** 从 2 开始枚举，第一个整除 n 的 i 就是最小质因子，`n/i` 即较大质因子。

---

### 14. 回文质数

**题意：** 输出 [a, b] 内所有既是回文数又是质数的数。

**关键结论：** 偶数位的回文数（除11外）都能被11整除，所以只需枚举1、2、3、5、7位的回文数。

```cpp
// 1位：枚举奇数 1,3,5,7,9
// 2位：只有 11
// 3位：枚举 d1d2d1（d1为奇数）
for (int d1=1; d1<=9; d1+=2)
    for (int d2=0; d2<=9; d2++)
        check(100*d1 + 10*d2 + d1, a, b);
// 5位、7位类似构造
```

**要点：** 构造回文数而非枚举所有数再判断，效率更高；d1 必须是奇数（偶数开头的大回文数不是质数）。

## 四、数字操作

### 15. 计数问题

**题意：** 1 到 n 中，数字 x（0-9）一共出现多少次。

```cpp
for (int i=1; i<=n; i++) {
    int num = i;
    while (num > 0) { if (num%10 == x) cnt++; num/=10; }
}
```

**模板：** 逐位提取 → `num%10` 取个位，`num/=10` 移位。

---

### 16. 数字反转

**题意：** 输入整数（可能为负），输出反转后的整数（自动去前导零）。

```cpp
bool negative = (s[0] == '-');
string num = negative ? s.substr(1) : s;
reverse(num.begin(), num.end());
cout << (negative?"-":"") << stoi(num);  // stoi 自动去前导零
```

**要点：** `reverse` 反转字符串；`stoi` 将字符串转为整数，自动去除前导零。

---

### 17. 斐波那契数列

**题意：** 求第 n 项斐波那契数（保留2位小数，用公式）。

```cpp
double sq5 = sqrt(5.0);
double a = (1+sq5)/2, b = (1-sq5)/2;
printf("%.2f", (pow(a,n) - pow(b,n)) / sq5);  // 比内特公式
```

**递推写法（更常用）：**

```cpp
long long f[N]; f[1]=1; f[2]=1;
for (int i=3; i<=n; i++) f[i] = f[i-1] + f[i-2];
```

## 五、图案输出

### 18. 数字直角三角形

**题意：** n 行，第 i 行有 n-i+1 个数，连续编号从1开始，两位补零。

```cpp
int num = 1;
for (int i=1; i<=n; i++) {
    for (int j=0; j<n-i+1; j++) printf("%02d", num++);
    cout << "\n";
}
```

**要点：** `printf("%02d", x)` 两位补零；内层循环长度 `n-i+1` 随行数递减。

---

### 19. 求三角形

**题意：** 输出 n×n 的正方形数字阵 + 空行 + 右对齐的三角形数字阵，两位补零。

```cpp
// 正方形：n行×n列
// 三角形：第i行前有 (n-i)*2 个空格，然后 i 个数
for (int i=1; i<=n; i++) {
    for (int k=0; k<(n-i)*2; k++) cout << " ";  // 前导空格
    for (int j=0; j<i; j++) printf("%02d", num2++);
    cout << "\n";
}
```

**要点：** 每个数字占 2 字符，所以空格也要补 `(n-i)*2` 个来对齐。

## 六、最长子序列

### 20. 最长连号

**题意：** 找最长的连续递增1的子序列长度（如 3 4 5 长度为3）。

```cpp
int maxLen=1, curLen=1;
for (int i=1; i<n; i++) {
    if (a[i] == a[i-1]+1) curLen++;
    else                   curLen=1;  // 断了就重置
    maxLen = max(maxLen, curLen);
}
```

**要点：** `maxLen` 在每次迭代都更新，不能只在 `else` 时更新。

## 七、枚举

### 21. Davor

**题意：** 给 N，每周存款总额 M = N/52，M = 7X + 3K（X 份每份7元，K 份每份3元），从大到小枚举 X，找第一个合法解。

```cpp
int M = N / 52;
for (int X=100; X>=1; X--) {
    int K = (M/7 - X) / 3;  // 由 X + 3K = M/7 推出
    if (K>0 && X + 3*K == M/7) { cout<<X<<"\n"<<K; break; }
}
```

**要点：** 题目要求 X 尽量大，从大到小枚举，找到第一个合法解即停；注意验证等式 `X + 3K == M/7`。

## 循环结构通用思路

```
明确：循环对象是什么？（数字、下标、状态）
  ↓
选择循环类型：
  for  → 已知次数或范围确定
  while → 不知道循环几次，按条件终止
  ↓
每轮做什么？（累加、更新最值、输出、修改状态）
  ↓
循环后还需要做什么？（输出最后一段、计算平均值）
```

## 常用模板速查

| 场景            | 写法                                                 |
| --------------- | ---------------------------------------------------- |
| 1到n求和        | `for(i=1;i<=n;i++) sum+=i`                         |
| 逐位提取数字    | `while(num>0){digit=num%10; num/=10;}`             |
| 维护最大/最小值 | 初始化为 `INT_MIN`/`INT_MAX` 或 `-1e9`/`1e9` |
| 质数判断        | `for(i=2; i*i<=n; i++) if(n%i==0) return false`    |
| 两位补零输出    | `printf("%02d", x)`                                |
| 字符串反转      | `reverse(s.begin(), s.end())`                      |
| 字符串转整数    | `stoi(s)`（自动去前导零）                          |
| 蛇形序列累计    | 用 `while(k>0)` 按阶段消耗                         |

## 易错点

1. **循环边界**：`i<=n` vs `i<n`，差一个元素
2. **while 循环后收尾**：游程编码/级数求和的最后一段不会被循环内代码处理，须循环后补输出
3. **整数除法**：`1/n = 0`，需写 `1.0/n`
4. **`i*i <= n` 溢出**：当 n 较大时 `i*i` 可能溢出 int，改用 `long long` 或 `i <= n/i`
5. **最大值初始化**：初始化为 `0` 可能导致全负数数组结果错误，应初始化为 `INT_MIN` 或第一个元素
6. **printf 格式**：`%.2f` 用于 `double`，`%lld` 用于 `long long`
