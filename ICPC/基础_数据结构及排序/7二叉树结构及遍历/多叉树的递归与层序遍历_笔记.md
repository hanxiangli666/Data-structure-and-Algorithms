# 多叉树的递归/层序遍历 学习笔记

> **前置知识**：二叉树的递归遍历和层序遍历  
> **核心认知**：多叉树就是二叉树的自然延伸——把「左右两个孩子」变成「任意多个孩子」，其他一模一样  
> **新概念**：森林 = 多棵多叉树的集合

---

## 一、从二叉树到多叉树：只有一处改动

### 1.1 节点结构对比

```cpp
// 二叉树节点：固定 2 个子节点
class TreeNode {
public:
    int val;
    TreeNode* left;      // 左孩子
    TreeNode* right;     // 右孩子
};

// 多叉树节点：任意个子节点
class Node {
public:
    int val;
    vector<Node*> children;   // 所有孩子放在一个数组里
};
```

**就这点区别，其他全部一样。**

**类比**：二叉树像一个「每个人最多生两个孩子」的家庭；多叉树像「每个人可以生任意多个孩子」的家庭。家族的遍历方式（挨个拜访）并没有本质变化，只是每家要拜访的孩子数量不同。

### 1.2 图示对比

```
二叉树：                    多叉树：
      1                         1
     / \                     /  |  \
    2   3                   2   3   4
   /   / \                 /|\     / \
  4   5   6               5 6 7   8   9
```

## 二、递归遍历（DFS）

### 2.1 框架对比

```cpp
// 二叉树 DFS
void traverse(TreeNode* root) {
    if (root == nullptr) return;
    // 【前序位置】
    traverse(root->left);
    // 【中序位置】
    traverse(root->right);
    // 【后序位置】
}

// 多叉树 DFS
void traverse(Node* root) {
    if (root == nullptr) return;
    // 【前序位置】
    for (Node* child : root->children) {
        traverse(child);
    }
    // 【后序位置】
}
```

**两个变化**：

| 变化 | 二叉树 | 多叉树 |
|------|--------|--------|
| 递归调用 | 写死两行：`traverse(left)` + `traverse(right)` | 一个 for 循环遍历所有孩子 |
| 中序位置 | 有（左子树和右子树之间） | **没有**（孩子有多个，"中间"没有意义） |

**C++ 知识点**：
- `for (Node* child : root->children)`：范围 for 循环（range-based for），遍历 vector 中的每个元素
- 相当于 Python 的 `for child in root.children`

### 2.2 前序遍历示例

```
多叉树：
        1
      / | \
     2  3  4
    /|     |
   5 6     7
```

```cpp
void traverse(Node* root) {
    if (root == nullptr) return;
    cout << root->val << " ";          // 前序位置：进入节点时记录
    for (Node* child : root->children) {
        traverse(child);
    }
}
```

遍历过程（和二叉树一样，先深入到底再回头）：

```
进入 1 → 记录 1
  进入 2 → 记录 2
    进入 5 → 记录 5（叶子，返回）
    进入 6 → 记录 6（叶子，返回）
  返回到 1
  进入 3 → 记录 3（叶子，返回）
  进入 4 → 记录 4
    进入 7 → 记录 7（叶子，返回）
  返回到 1

前序结果：[1, 2, 5, 6, 3, 4, 7]
```

### 2.3 后序遍历示例

```cpp
void traverse(Node* root) {
    if (root == nullptr) return;
    for (Node* child : root->children) {
        traverse(child);
    }
    cout << root->val << " ";          // 后序位置：所有孩子遍历完才记录
}
```

```
后序结果：[5, 6, 2, 3, 7, 4, 1]
```

每个节点都是在它所有子节点遍历完之后才被记录，所以根节点 1 最后出现。

## 三、层序遍历（BFS）

和二叉树完全一样，只是把「加入左右孩子」换成「遍历所有孩子加入队列」。

### 3.1 写法一：最简单

```cpp
void levelOrderTraverse(Node* root) {
    if (root == nullptr) return;
    
    queue<Node*> q;
    q.push(root);
    
    while (!q.empty()) {
        Node* cur = q.front();
        q.pop();
        
        cout << cur->val << " ";
        
        // 二叉树：加入左右孩子
        // 多叉树：遍历所有孩子加入
        for (Node* child : cur->children) {
            q.push(child);
        }
    }
}
```

改动只有一处：把 `if (left) push(left); if (right) push(right);` 换成了 `for (child : children) push(child);`

### 3.2 写法二：记录层数（最常用）

```cpp
void levelOrderTraverse(Node* root) {
    if (root == nullptr) return;
    
    queue<Node*> q;
    q.push(root);
    int depth = 1;
    
    while (!q.empty()) {
        int sz = q.size();       // 当前层的节点数
        for (int i = 0; i < sz; i++) {
            Node* cur = q.front();
            q.pop();
            
            cout << "depth=" << depth << " val=" << cur->val << endl;
            
            for (Node* child : cur->children) {
                q.push(child);
            }
        }
        depth++;
    }
}
```

和二叉树的写法二一模一样的模式。

### 3.3 写法三：每个节点自带状态

```cpp
struct State {
    Node* node;
    int depth;
    State(Node* n, int d) : node(n), depth(d) {}
};

void levelOrderTraverse(Node* root) {
    if (root == nullptr) return;
    
    queue<State> q;
    q.push(State(root, 1));
    
    while (!q.empty()) {
        State cur = q.front();
        q.pop();
        
        cout << "depth=" << cur.depth << " val=" << cur.node->val << endl;
        
        for (Node* child : cur.node->children) {
            q.push(State(child, cur.depth + 1));
        }
    }
}
```

## 四、新概念：森林

### 4.1 什么是森林？

**森林 = 多棵树的集合**。

```
  树1       树2       树3
   1         5         8
  / \       /|\        |
 2   3     6 7 9      10
 |
 4
```

用代码表示就是一个根节点的列表：

```cpp
vector<Node*> forest;   // 每个元素是一棵树的根节点
```

**单独一棵树也是一个特殊的森林**（只有一个根节点的森林）。

### 4.2 遍历森林

对每棵树分别遍历即可：

```cpp
// DFS 遍历整个森林
for (Node* root : forest) {
    traverse(root);
}

// BFS 遍历整个森林
for (Node* root : forest) {
    levelOrderTraverse(root);
}
```

### 4.3 森林在哪里用到？

后面学**并查集（Union-Find）**时会用到。并查集维护的就是一个森林——多棵树，每棵树的根节点代表一个集合。现在先知道这个概念就行。

## 五、二叉树 → 多叉树 改动速查表

| 操作 | 二叉树写法 | 多叉树写法 |
|------|-----------|-----------|
| 节点定义 | `left`, `right` 两个指针 | `vector<Node*> children` |
| DFS 递归调用 | `traverse(left); traverse(right);` | `for (child : children) traverse(child);` |
| 中序位置 | 有 | **无** |
| BFS 入队 | `if(left) push(left); if(right) push(right);` | `for (child : children) push(child);` |
| 其他逻辑 | — | **完全一样** |
