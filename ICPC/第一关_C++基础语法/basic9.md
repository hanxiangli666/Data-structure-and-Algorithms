# 拆解步骤

**第一步：怎么读入这个字符串？**

输入是 `HeLLo WorLD`，中间有空格，所以不能用 `cin >>`（它遇到空格会停）。

要用什么？想想我刚才讲的。`cin >>` 读到空格就停下来，所以只会读到 `HeLLo`，后面的 ` WorLD` 就丢了。读含空格的一整行，只有一个办法。

---

**第二步：怎么遍历字符串里每个字符？**

我刚才讲过遍历字符串的写法：

```cpp
for (char c : s) {
    // 对每个字符做点什么
}
```

注意循环变量类型是 `char`，不是 `int`——字符串里存的是字符，不是整数。

---

**第三步：怎么判断一个字符是大写？**

我刚才也讲过，有一个现成的函数可以直接用，叫什么来着？不用自己判断 ASCII 范围，标准库帮你做好了，一个函数调用搞定。

---

**第四步：怎么计数？**

定义一个变量 `count = 0`，每发现一个大写字母就 `count++`，最后输出。这是竞赛里最常见的"计数"模式，遇到"统计多少个……"的题，第一反应就是这个结构。

> 💡 **老师提醒：** `count` 这个变量名在 C++ 标准库里有同名函数（`std::count`），加了 `using namespace std` 后偶尔会引发命名冲突。竞赛里通常没问题，但如果遇到奇怪的编译错误，可以把变量名改成 `cnt`——这也是竞赛选手的惯用写法。

---

# 把四步串起来

```
读入字符串
→ 遍历每个字符
  → 如果是大写字母
    → count加一
→ 输出count
```
