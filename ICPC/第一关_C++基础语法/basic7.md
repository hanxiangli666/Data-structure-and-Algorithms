# 你的代码问题

**问题一：函数参数没有写类型**

```cpp
int maxOfThree(a, b, c) {      // ❌ 错误
int maxOfThree(int a, int b, int c) {  // ✅ 正确
```

C++必须给每个参数声明类型，不像Python可以省略。C++ 是静态类型语言，编译器在编译阶段就要确定每个参数的大小和内存布局——不给类型，编译器根本没法生成对应的汇编代码。

**问题二：`max()` 不支持三个参数**

```cpp
return max(a, b, c);    // ❌ C++的max只能比较两个数
return max(a, max(b, c));  // ✅ 嵌套调用
```

C++ 标准库的 `max` 函数是模板实现的，只接收两个同类型参数。Python 的 `max()` 能接收任意多个参数，是因为 Python 在底层做了特殊处理；C++ 的 `max` 没有这个机制，嵌套调用是标准写法：先比 `b` 和 `c` 取大者，再和 `a` 比。

> 💡 **老师提醒：** C++17 引入了 `std::max({a, b, c})` 的写法，用初始化列表可以比较多个值。但嵌套的 `max(a, max(b, c))` 更通用，竞赛里两种都能用，建议先记嵌套写法。

**问题三：函数体的花括号没有关闭**

```cpp
int maxOfThree(int a, int b, int c) {
    return max(a, max(b, c));
}   // ← 这个花括号你漏掉了
```

**问题四：`main` 函数缺少花括号和 return**

```cpp
int main() {
    cin >> a >> b >> c;
    cout << maxOfThree(a, b, c) << endl;
}   // ← 漏掉了
```

**问题五：全局变量不必要**

你把 `a, b, c` 定义在了函数外面（全局变量），这样其实可以运行，但是不好的习惯。变量应该定义在 `main` 里面。全局变量的问题在于：它的生命周期贯穿整个程序，任何函数都能修改它，一旦代码量多了起来，很难追踪变量被谁改动了，极容易引入隐蔽的 bug。

# 修正后的完整代码

```cpp
#include <iostream>
using namespace std;

int maxOfThree(int a, int b, int c) {
    return max(a, max(b, c));
}

int main() {
    int a, b, c;
    cin >> a >> b >> c;
    cout << maxOfThree(a, b, c) << endl;
    return 0;
}
```
