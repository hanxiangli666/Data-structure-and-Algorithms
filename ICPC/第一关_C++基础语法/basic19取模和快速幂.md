好，我们退一步，把这两个概念单独讲清楚。

# 先搞清楚取模 `%` 在干什么

`%` 就是 **求余数** ：

```
10 % 3 = 1    （10除以3，商3余1）
7  % 3 = 1    （7除以3，商2余1）
6  % 3 = 0    （6除以3，商2余0）
```

对应Python的 `%` 完全一样，这个你应该懂。

---

# 为什么竞赛题要对1e9+7取模？

因为答案可能 **大到存不下** 。

举个例子，计算 $2^{100}$：

```
2^10  = 1024
2^20  = 1048576
2^30  = 1073741824
2^60  ≈ 1.15 × 10^18   ← long long 快装不下了
2^100 ≈ 1.27 × 10^30   ← 彻底装不下，溢出！
```

`long long` 最大只能存到约 $9.2 \times 10^{18}$，超过就溢出，答案就错了。

所以题目说：**你不需要输出真实答案，只需要输出答案除以1e9+7的余数。**

1e9+7 是一个质数，选质数作为模数有数学上的好处（可以方便地求模逆元），所以它成了竞赛里的标准模数。你不需要深究为什么是质数，记住这个数字就行。

> 💡 **老师提醒：** 有些题用的是 `998244353`，同样是质数，和 `1e9+7` 可以互换用。看到题目说"对某个大质数取模"，处理方式完全一样。

# 取模的核心性质

这是关键，记住这三条：

```cpp
(a + b) % m = ((a % m) + (b % m)) % m
(a - b) % m = ((a % m) - (b % m) + m) % m
(a * b) % m = ((a % m) * (b % m)) % m
```

**直觉理解：** 你可以在每一步计算后取模，最终结果和最后再取模是一样的。

举个例子，计算 $(3 \times 4 \times 5)$ 对 $7$ 取模：

```
方法一：先全部算出来再取模
3 × 4 × 5 = 60
60 % 7 = 4

方法二：每步都取模
3 × 4 = 12，12 % 7 = 5
5 × 5 = 25，25 % 7 = 4

两种方法答案一样！
```

所以计算大数时， **每步乘法后立刻取模** ，数字就永远不会太大：

```cpp
const int MOD = 1e9 + 7;

long long result = 1;
result = result * 3 % MOD;   // 每步取模
result = result * 4 % MOD;   // 数字始终在MOD范围内
result = result * 5 % MOD;
```

注意减法需要特殊处理：`(a - b + MOD) % MOD`，加上 MOD 是为了防止出现负数。*就像取模和溢出的关系，就像一个只能装1000毫升的水杯——你不能等装满了再倒，得每加一点水就先倒掉超出的部分，确保杯子永远不满。*

> ✅ 取模三条核心公式记住了，以后遇到需要取模的题，每步操作后无脑加 `% MOD`，就不会溢出！

# 现在来讲快速幂

## 朴素方法（太慢）

计算 $2^{10}$，最直接的方法：

```cpp
long long result = 1;
for (int i = 0; i < 10; i++) {
    result = result * 2 % MOD;
}
// 循环10次
```

如果 b = 1000000000（十亿），循环十亿次，直接超时。

---

## 快速幂的直觉

计算 $2^8$，你会怎么做？

```
笨方法：2×2×2×2×2×2×2×2，乘7次

聪明方法：
2^1 = 2
2^2 = 2^1 × 2^1 = 4        （1次乘法）
2^4 = 2^2 × 2^2 = 16       （1次乘法）
2^8 = 2^4 × 2^4 = 256      （1次乘法）

只用3次乘法！
```

**核心思路：每次把指数折半，底数平方。**

---

## 如果指数是奇数怎么办？

比如 $2^{11}$：

```
11 是奇数，先拆出一个2：
2^11 = 2^1 × 2^10

2^10 继续折半：
2^10 = 2^5 × 2^5

2^5 是奇数，再拆：
2^5 = 2^1 × 2^4

2^4 继续折半：
2^4 = 2^2 × 2^2

2^2 = 2^1 × 2^1
```

用二进制看更直觉：

```
11 = 1011（二进制）

2^11 = 2^8 × 2^2 × 2^1
         ↑     ↑     ↑
      第3位  第1位  第0位（从0开始数）
```

---

## 代码逐行解释

```cpp
long long power(long long a, long long b, long long mod) {
    long long result = 1;  // 最终答案，初始为1
    a %= mod;              // 先对底数取模

    while (b > 0) {
        if (b & 1) {                    // 如果b的最低位是1（b是奇数）
            result = result * a % mod;  // 把当前的a乘进结果
        }
        a = a * a % mod;  // 底数平方
        b >>= 1;          // 指数除以2（右移一位）
    }

    return result;
}
```

我们用 $2^{11}$ 手动跑一遍：

```
初始：result=1, a=2, b=11(1011)

第1轮：
  b=11，b&1=1（奇数），result = 1×2 = 2
  a = 2×2 = 4
  b = 11>>1 = 5(101)

第2轮：
  b=5，b&1=1（奇数），result = 2×4 = 8
  a = 4×4 = 16
  b = 5>>1 = 2(10)

第3轮：
  b=2，b&1=0（偶数），result不变 = 8
  a = 16×16 = 256
  b = 2>>1 = 1(1)

第4轮：
  b=1，b&1=1（奇数），result = 8×256 = 2048
  a = 256×256
  b = 1>>1 = 0

b=0，循环结束，返回2048

验证：2^11 = 2048 ✅
```

> 💡 **老师提醒：** 快速幂函数里 `a %= mod` 这一步必须在循环之前做，否则第一轮 `a * a` 就可能溢出。另外 `result` 的初始值必须是 `1`（不能是 `0`），因为任何数乘以 1 还是它本身，而乘以 0 就全清零了。

# 总结

```
取模：让数字永远不超出范围
      每步乘法后立刻 % MOD

快速幂：把 O(b) 次乘法压缩到 O(log b) 次
        核心：指数折半，底数平方
        奇数指数：先把当前底数乘进结果
```
