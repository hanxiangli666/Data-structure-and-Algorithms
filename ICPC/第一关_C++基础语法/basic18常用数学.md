## 一、取模运算

竞赛题经常说"答案对1e9+7取模"，这是因为答案可能非常大，超出int/long long的范围。

```cpp
const int MOD = 1e9 + 7;  // 竞赛中最常见的模数

// 基本取模
cout << 10 % 3 << endl;   // 1

// 加法取模
(a + b) % MOD

// 乘法取模
(a * b) % MOD

// 减法取模（注意可能出现负数！）
(a - b + MOD) % MOD   // 加MOD防止负数
```

**为什么加法减法要取模，乘法也要取模？**

```cpp
// 假设 a = 1e9, b = 1e9
// a + b = 2e9，超出int范围（int最大约2.1e9，很危险）
// 所以每步运算后都要取模

long long a = 1e9, b = 1e9;
long long result = (a + b) % MOD;  // 安全
```

## 二、long long

当数字可能超过21亿时，用 `long long`代替 `int`：

```cpp
int max       ≈ 2.1 × 10^9
long long max ≈ 9.2 × 10^18
```

```cpp
long long a = 1000000000LL;  // 后面加LL表示这是long long字面量
long long b = a * a;         // 如果用int，这里会溢出！
```

**竞赛习惯：** 看到数据范围超过1e9，立刻换long long。

```cpp
// 很多竞赛选手直接这样写，省去思考用int还是long long
#define ll long long

ll a, b;
cin >> a >> b;
```

## 三、快速幂

计算 $a^b$ 最朴素的方法是连乘b次，但b可能很大（比如1e9），会超时。

**快速幂** 利用以下规律：

```
a^8 = a^4 * a^4          （拆成两半）
a^4 = a^2 * a^2
a^2 = a^1 * a^1

所以 a^8 只需要3次乘法，而不是7次
```

规律：

* b是偶数：$a^b = (a^{b/2})^2$
* b是奇数：$a^b = a \times a^{b-1}$

代码：

```cpp
long long power(long long a, long long b, long long mod) {
    long long result = 1;
    a %= mod;
    while (b > 0) {
        if (b & 1) result = result * a % mod;  // b是奇数
        a = a * a % mod;                        // a平方
        b >>= 1;                                // b除以2
    }
    return result;
}

// 使用
cout << power(2, 10, MOD) << endl;  // 2^10 = 1024
```

时间复杂度从 $O(b)$ 降到 $O(\log b)$，快很多。

> 这段代码实现的是 **快速幂（Exponentiation by Squaring）** 算法。它的核心思想是：**与其老老实实地连乘 **$n$** 次，不如利用“平方”来成倍地加速。**
>
> 我们可以把 **$2^{10}$** 想象成一个搬运任务。普通方法要搬 10 次，而快速幂通过“翻倍”只需要 4 步左右。
>
> ---
>
> ### 1. 核心逻辑：拆解指数
>
> 任何一个数字的指数（比如 10）都可以拆成 2 的幂之和（其实就是写成二进制）。
>
> 对于 **$2^{10}$**：
>
> * **$10$** 的二进制是 `1010`。
> * 也就是说 **$10 = 8 + 2$**。
> * 那么 **$2^{10} = 2^8 \times 2^2$**。
>
> 代码中的 `b & 1` 就是在检查二进制的每一位：如果是 1，就把当前的“能量包”乘进结果里。
>
> ---
>
> ### 2. 举个例子：计算 **$2^{10}$**（假设 mod 很大）
>
> 我们来看变量 `a`（底数）、`b`（指数）和 `result`（结果）的变化过程：
>
> | **轮次**    | **指数 b (二进制)** | **b & 1 (是否为奇数/末位为1)** | **result (累积结果)**       | **a (当前的底数能量包)** | **动作**                                  |
> | ----------------- | ------------------------- | ------------------------------------ | --------------------------------- | ------------------------------ | ----------------------------------------------- |
> | **初始**    | 10 (1010)                 | -                                    | 1                                 | **$2^1$**= 2           | 准备开始                                        |
> | **第 1 轮** | 10 (1010)                 | **否**(0)                      | 1                                 | **$2^2$**= 4           | **$ b$**变 5，底数翻倍                        |
> | **第 2 轮** | 5 (0101)                  | **是**(1)                      | **$1 \times 4 = 4$**      | **$2^4$**= 16          | **乘入 result** ，$**b$变 2，底数再翻倍 |
> | **第 3 轮** | 2 (0010)                  | **否**(0)                      | 4                                 | **$2^8$**= 256         | **$b$**变 1，底数再翻倍                       |
> | **第 4 轮** | 1 (0001)                  | **是**(1)                      | **$4 \times 256 = 1024$** | **$2^{16}$**= ...      | **乘入 result** ，**$b$**变 0，结束     |
>
> ---
>
> ### 3. 代码每一行在干什么？
>
> * **`a %= mod;`** ：先取模，防止底数太大直接爆掉。
> * **`if (b & 1)`** ：判断 `b` 的二进制最后一位是不是 1。如果是，说明这一位的“重量”需要计入结果。
> * **`a = a * a % mod;`** ：这是算法的精髓！不管本轮要不要计入结果，底数 `a` 都在不断 **自我进化（平方）** 。它依次变成 **$a^1, a^2, a^4, a^8, a^{16} \dots$**。
> * **`b >>= 1;`** ：把 `b` 的二进制向右移一位，相当于 `b = b / 2`。
>
> ---
>
> ### 4. 为什么要这么写？
>
> * **普通连乘** ：计算 **$a^{10^9}$** 需要乘 10 亿次。
> * **快速幂** ：计算 **$a^{10^9}$** 只需要大约 **$\log_2(10^9) \approx 30$** 次计算。
>
> **它把“线性级”的时间复杂度降到了“对数级”** ，是竞赛和加密算法（如 RSA）的基础。
>
> 这个“翻倍”的逻辑清楚了吗？如果不清楚，我可以再换个“折纸”的比喻帮你理解！你想试试计算一个更大的数（比如 **$3^5 \pmod{10}$**）来练练手吗？

## 四、最大公约数 GCD

```cpp
// C++17 内置了gcd，直接用
#include <numeric>
cout << __gcd(12, 8) << endl;   // 4
cout << gcd(12, 8) << endl;     // 4（C++17）

// 最小公倍数
int a = 12, b = 8;
cout << a / gcd(a, b) * b << endl;  // 24
```

手写版（万一编译器不支持）：

```cpp
int gcd(int a, int b) {
    return b == 0 ? a : gcd(a, b % a);
}
```

`?:` 是三元运算符，等价于：

```cpp
if (b == 0) return a;
else return gcd(a, b % a);
```

## 五、常用数学函数

```cpp
#include <cmath>

abs(-5)      // 绝对值：5（int版本）
fabs(-3.14)  // 绝对值：3.14（double版本）
sqrt(16.0)   // 平方根：4.0
pow(2, 10)   // 幂次：1024.0（返回double，竞赛中慎用，有精度问题）
ceil(3.2)    // 向上取整：4.0
floor(3.8)   // 向下取整：3.0
```

⚠️ 竞赛中能用整数就不用浮点数，浮点数有精度问题。比如求幂次用快速幂，不用 `pow`。
