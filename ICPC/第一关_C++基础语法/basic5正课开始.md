# 第一课：输入输出

## 基本输出 cout（你已经会了）

```cpp
cout << "Hello" << endl;
cout << 123 << endl;
```

## 基本输入 cin

```cpp
#include <iostream>
using namespace std;

int main() {
    int a;
    cin >> a;        // 读取一个整数
    cout << a << endl;
    return 0;
}
```

`cin >> a` 就是"从键盘读一个值，存进 a"。`>>` 是"提取运算符"，方向是从 `cin`（键盘流）往变量里"倒"数据，和 `<<` 输出时的方向正好相反。*就像 `>>` 和 `<<` 的方向，就像箭头指向哪里，数据就往哪里走——方向看对了，读和写就不会搞混。*

对应Python的：

```python
a = int(input())
```

> 💡 **老师提醒：** C++ 的 `cin >>` 默认会**跳过空白字符**（空格、回车、Tab）再读值，所以一行里写 `3 5` 或分两行写 `3\n5`，`cin >> a >> b` 都能正确读取。这和 Python 的 `input()` 不一样——Python 一次读一整行，C++ 的 `cin` 是按"词"读的。

**读取多个值：**

```cpp
int a, b;
cin >> a >> b;       // 读两个整数，用空格或回车分隔都行
```

```cpp
string s;
cin >> s;            // 读一个单词（遇到空格停止）
```

## 竞赛中最常见的输入模式

**模式一：读固定数量**

```
输入：
5
1 2 3 4 5
```

```cpp
int n;
cin >> n;

vector<int> a(n);
for (int i = 0; i < n; i++) {
    cin >> a[i];
}
```

先读 `n`，再用 `n` 控制循环次数——这是竞赛里最经典的读数组方式，90% 的题都长这个样子，直接背下来。

**模式二：读到文件结尾（EOF）**

```
输入：
1 2
3 4
5 6
（不知道有多少行）
```

```cpp
int a, b;
while (cin >> a >> b) {    // 读不到就退出循环
    cout << a + b << endl;
}
```

`while (cin >> a >> b)` 之所以能这样写，是因为 `cin >> a >> b` 读取成功时返回 `cin` 本身，而 `cin` 在布尔上下文里是 `true`；读到文件末尾失败时变为 `false`，循环自然退出。

> 💡 **老师提醒：** `while(cin >> a >> b)` 只在"不知道输入有多少组"时用。如果题目明确说了输入几行，就用普通的 `for` 循环配合固定次数的 `cin`，不要所有题都套 `while`。

**模式三：读整行字符串**

```cpp
string line;
getline(cin, line);    // 读一整行，包括空格
```

`cin >>` 读到空格就停，但字符串里往往含有空格，这时就必须用 `getline`。记住这个区别，后面字符串那课会专门讲。

> ✅ 三种输入模式记住了，竞赛里 80% 的题输入都在这三类里，读题时先判断是哪种，再套对应的写法！
