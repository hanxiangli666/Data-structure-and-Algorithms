# 第一课：输入输出

## 基本输出 cout（你已经会了）

```cpp
cout << "Hello" << endl;
cout << 123 << endl;
```

## 基本输入 cin

```cpp
#include <iostream>
using namespace std;

int main() {
    int a;
    cin >> a;        // 读取一个整数
    cout << a << endl;
    return 0;
}
```

`cin >> a` 就是"从键盘读一个值，存进 a"。

对应Python的：

```python
a = int(input())
```

**读取多个值：**

```cpp
int a, b;
cin >> a >> b;       // 读两个整数，用空格或回车分隔都行
```

```cpp
string s;
cin >> s;            // 读一个单词（遇到空格停止）
```

---

## 竞赛中最常见的输入模式

**模式一：读固定数量**

```
输入：
5
1 2 3 4 5
```

```cpp
int n;
cin >> n;

vector<int> a(n);
for (int i = 0; i < n; i++) {
    cin >> a[i];
}
```

**模式二：读到文件结尾（EOF）**

```
输入：
1 2
3 4
5 6
（不知道有多少行）
```

```cpp
int a, b;
while (cin >> a >> b) {    // 读不到就退出循环
    cout << a + b << endl;
}
```

**模式三：读整行字符串**

```cpp
string line;
getline(cin, line);    // 读一整行，包括空格
```
