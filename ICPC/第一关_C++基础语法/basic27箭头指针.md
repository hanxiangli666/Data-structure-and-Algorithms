`->` 就是指针版本的 `.`，一句话就能说清楚。

---

# `.` 和 `->` 的区别

```cpp
// 普通变量用 .
Student s;
s.name = "Alice";    // s是普通变量，用 .

// 指针用 ->
Student *p = &s;
p->name = "Alice";   // p是指针，用 ->
```

**规律：**

```
变量  → 用 .
指针  → 用 ->
```

---

# `->` 其实是两步的缩写

```cpp
p->name
// 等价于
(*p).name   // 先解引用，再访问成员
```

`->` 只是把这两步合并成一个符号，更简洁。

---

# 什么时候会遇到指针？

```cpp
// 1. 用new创建对象时
Student *p = new Student("Alice", 20);
p->name;       // 用 ->
p->introduce(); // 用 ->

// 2. this指针（在类内部）
this->name = n;  // this是指针，用 ->

// 3. 链表/树节点（后面学到会遇到）
struct Node {
    int val;
    Node *next;   // 指向下一个节点的指针
};

Node *cur = head;
cur->val;         // 用 ->
cur->next;        // 用 ->
```

---

# 总结

```
. 用于普通变量
-> 用于指针

看到 -> 就知道左边一定是个指针
```

竞赛中 `->` 主要在链表和树的题目里出现，现阶段了解即可。
