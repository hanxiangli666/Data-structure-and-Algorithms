# 四步走

**第一步：读入数据**

```
读一个整数 n
读 n 个整数存进 vector
```

这个你在第一课学过，模式一：读固定数量。先读 `n`，再用 `vector<int> a(n)` 开好空间，然后循环 `cin >> a[i]`。

---

**第二步：排序**

刚才讲过，一行代码，用 `sort`。`sort(a.begin(), a.end())` 默认从小到大，正是这题需要的。

---

**第三步：去重**

刚才讲过，两行代码，`unique` + `erase`。记住顺序不能反，而且必须先排序 `unique` 才有效——这两步刚好接在第二步之后。

---

**第四步：输出**

遍历 vector，用 `cout` 打印每个元素。题目要求输出 `1 2 3`，注意是空格分隔，不是每个数占一行——用 `cout << x << " "` 而不是 `cout << x << endl`。

> 💡 **老师提醒：** 去重后 vector 的大小已经变了，不再是原来的 `n`。遍历输出时用范围 for 循环 `for (int x : a)` 最保险，不用操心大小问题。
