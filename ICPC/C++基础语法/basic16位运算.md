# 第六课：位运算

## 先理解"位"是什么

计算机里所有数字都是用**二进制**存的，比如：

```
十进制  二进制
  1  →  0001
  2  →  0010
  3  →  0011
  4  →  0100
  5  →  0101
```

位运算就是 **直接对二进制的每一位进行操作** ，速度极快。

---

## 六种位运算

### 1. 按位与 `&`

两个位都是1才是1，否则是0：

```
  5 = 0101
  3 = 0011
  & = 0001 = 1
```

```cpp
cout << (5 & 3) << endl;  // 1
```

**最常用场景：判断奇偶**

```cpp
if (n & 1) {
    cout << "奇数" << endl;
} else {
    cout << "偶数" << endl;
}
// 比 n % 2 更快
```

---

### 2. 按位或 `|`

两个位有一个是1就是1：

```
  5 = 0101
  3 = 0011
  | = 0111 = 7
```

```cpp
cout << (5 | 3) << endl;  // 7
```

---

### 3. 按位异或 `^`

两个位**不同**则为1，相同则为0：

```
  5 = 0101
  3 = 0011
  ^ = 0110 = 6
```

```cpp
cout << (5 ^ 3) << endl;  // 6
```

**最常用场景：找出只出现一次的数字**

```cpp
// 数组里只有一个数出现一次，其余都出现两次
// 相同的数异或结果为0，0异或任何数等于那个数
vector<int> v = {1, 2, 3, 2, 1};
int result = 0;
for (int x : v) {
    result ^= x;
}
cout << result << endl;  // 3
```

这是一道经典LeetCode题（136题），记住这个技巧！

>
> 这段代码利用了位运算中 **异或（XOR，符号为 `^`）** 的几个神奇特性。我们可以把它想象成一种“消消乐”或者“开关游戏”。
>
> 理解这段代码的关键在于掌握异或运算的三个核心性质：
>
> ### 1. 异或的核心性质
>
> * **归零性** ：**$a \oplus a = 0$**（相同的数相抵消，变回 0）。
> * **恒等性** ：**$a \oplus 0 = a$**（任何数和 0 异或，结果还是它自己）。
> * **交换律与结合律** ：**$a \oplus b \oplus a = a \oplus a \oplus b = 0 \oplus b = b$**。这意味着 **运算顺序不重要** ，只要成对出现的数字，最终都会互相抵消。
>
> ---
>
> ### 2. 过程拆解
>
> 拿你给的数组 `{1, 2, 3, 2, 1}` 来演示，程序执行的过程实际上是这样的：
>
> 1. **初始状态** ：`result = 0`
> 2. **遇到 1** ：`0 ^ 1 = 1`
> 3. **遇到 2** ：`1 ^ 2` (此时结果是 3，也就是二进制的 `01 ^ 10 = 11`)
> 4. **遇到 3** ：`3 ^ 3 = 0` (注意！这里的 3 是之前 1 和 2 的结合体，这一步比较抽象)
>
> **但如果我们根据“交换律”重新排列逻辑，就非常直观了：**
>
> $$
> 1 \oplus 2 \oplus 3 \oplus 2 \oplus 1
> $$
>
> $$
> \downarrow
> $$
>
> $$
> (1 \oplus 1) \oplus (2 \oplus 2) \oplus 3
> $$
>
> $$
> \downarrow
> $$
>
> $$
> 0 \oplus 0 \oplus 3 = 3
> $$
>
> 所有的“成对”数字都变成了 0，最后只剩下那个孤独的数字 `3` 和 `0` 做异或，结果依然是 `3`。
>
> ---
>
> ### 3. 为什么这比其他方法好？
>
> 如果你不用异或，你可能需要：
>
> * **先排序** ：时间复杂度 **$O(n \log n)$**。
> * **用哈希表计数** ：需要额外的空间来存储每个数字出现的次数。
>
>  **异或法的优势** ：
>
> * **快** ：只需要遍历一遍数组 (**$O(n)$**)。
> * **省** ：不需要额外空间 (**$O(1)$**)，只用一个变量 `result` 就搞定了。
>
>> **一句话总结** ：异或运算就像一个自带记忆的“去重滤镜”，它能自动把所有成对出现的数字过滤掉，最后剩下的就是那个唯一的单身数字。
>>
>
> 你想看看如果数组里有**两个**只出现一次的数字，该怎么用位运算解决吗？

### 4. 按位取反 `~`

每一位0变1，1变0：

```cpp
cout << (~5) << endl;  // -6（涉及补码，暂时记住结论）
// ~n = -(n+1)
```

竞赛中用得相对少，了解即可。

---

### 5. 左移 `<<`

把二进制整体向左移n位，相当于 **乘以2的n次方** ：

```
1 = 0001
1 << 1 = 0010 = 2   （乘2）
1 << 2 = 0100 = 4   （乘4）
1 << 3 = 1000 = 8   （乘8）
```

```cpp
cout << (1 << 3) << endl;  // 8
cout << (3 << 2) << endl;  // 12（3*4）
```

---

### 6. 右移 `>>`

把二进制整体向右移n位，相当于 **除以2的n次方** （取整）：

```
8 = 1000
8 >> 1 = 0100 = 4   （除2）
8 >> 2 = 0010 = 2   （除4）
```

```cpp
cout << (8 >> 1) << endl;  // 4
cout << (7 >> 1) << endl;  // 3（7/2=3.5，取整为3）
```

## 竞赛中最常用的位运算技巧

```cpp
// 1. 判断奇偶
n & 1          // 1是奇数，0是偶数

// 2. 乘以2 / 除以2
n << 1         // n * 2
n >> 1         // n / 2

// 3. 2的n次方
1 << n         // 2^n

// 4. 判断第k位是否为1（k从0开始）
(n >> k) & 1   // 1表示第k位是1，0表示是0

// 5. 把第k位设为1
n | (1 << k)

// 6. 把第k位设为0
n & ~(1 << k)
```

## 一个实用例子：枚举子集

竞赛中经常要枚举一个集合的所有子集，位运算可以很优雅地做到：

```cpp
// 集合 {0, 1, 2}，用二进制表示哪些元素被选中
// 000 = {} 
// 001 = {0}
// 010 = {1}
// 011 = {0,1}
// 100 = {2}
// 101 = {0,2}
// 110 = {1,2}
// 111 = {0,1,2}

int n = 3;
for (int mask = 0; mask < (1 << n); mask++) {
    cout << "子集：";
    for (int i = 0; i < n; i++) {
        if ((mask >> i) & 1) {  // 第i位是否为1
            cout << i << " ";
        }
    }
    cout << endl;
}
```

输出：

```
子集：
子集：0
子集：1
子集：0 1
子集：2
子集：0 2
子集：1 2
子集：0 1 2
```

这个技巧叫 **状态压缩** ，后面做竞赛题会经常用到。
